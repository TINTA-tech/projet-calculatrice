# projet-calculatrice

Projet de Programmation Mobile.

## Description
Application de calculatrice mobile développée avec Flutter, inspirée de la calculatrice Android.

## Fonctionnalités
- Addition
- Soustraction
- Multiplication
- Division
- Modulo (%)
- Gestion des nombres décimaux

## Technologies
- Flutter
- Dart
